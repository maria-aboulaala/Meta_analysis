Test de stationnarit (test de Dickey-Fuller augment)

Les hypothses de test de Dickey-Fuller augment (ADF) sont les suivantes :

  • Hypothse nulle (H0) : La srie temporelle possde une racine unitaire, ce qui signifie qu'elle n'est pas stationnaire.

  • Hypothse alternative (H1) : La srie temporelle ne possde pas de racine unitaire, ce qui signifie qu'elle est stationnaire.

Le test ADF vise valuer si la srie temporelle est stationnaire ou non. Pour effectuer ce test en Python, j'utilise la fonction adfuller de la bibliothque statsmodels.tsa.stattools. Cette fonction me permet de tester la stationnarit de la srie temporelle et d'obtenir les rsultats du test.

In[]:
from statsmodels.tsa.stattools import adfuller

result = adfuller(returns)
print("Test de stationnarit (Dickey-Fuller Augment):")
print(f"Statistique de test : {result[0]}")
print(f"Valeur critique  1% : {result[4]['1%']}")
print(f"Valeur critique  5% : {result[4]['5%']}")
print(f"Valeur critique  10% : {result[4]['10%']}")
print(f"P-value : {result[1]}")
print("\nRsultat : Les donnes sont stationnaires (p-value < 0.05)" if result[1] < 0.05 else "Rsultat : Les donnes ne sont pas stationnaires")

  • La p-value (2.1462624552861484e-16) est extrmement faible, bien en dessous du seuil de signification de 0.05. Cela indique que nous pouvons rejeter l'hypothse nulle (H0) selon laquelle la srie a une racine unitaire, ce qui signifie qu'elle n'est pas stationnaire.

Rsultat : Les donnes sont stationnaires.

Test de Normalit (Test de Shapiro-Wilk)

Le test de Shapiro-Wilk est un test statistique utilis pour valuer si un chantillon de donnes suit une distribution normale. Il est couramment utilis pour vrifier si les donnes sont normalement distribues avant d'appliquer certaines techniques statistiques. Voici les hypothses associes ce test :

  • Hypothse nulle (H0) : Les donnes suivent une distribution normale.

  • Hypothse alternative (H1) : Les donnes ne suivent pas une distribution normale.

En d'autres termes, le test de Shapiro-Wilk value si les donnes de l'chantillon proviennent d'une distribution normale ou non. Si la p-value du test est infrieure au niveau de signification choisi (gnralement = 0,05), on rejette l'hypothse nulle et on conclut que les donnes ne suivent pas une distribution normale.

Pour effectuer ce test, j'utilise le module scipy.stats. Cela me permet d'accder aux fonctions ncessaires pour effectuer le test de normalit.

In[]:
import scipy.stats as stats

result = stats.shapiro(returns)
print("Test de normalit (Shapiro-Wilk):")
print(f"Statistique de test : {result[0]}")
print(f"P-value : {result[1]}")
print("\nRsultat : Les donnes suivent une distribution normale (p-value > 0.05)" if result[1] > 0.05 else "Rsultat : Les donnes ne suivent pas une distribution normale")

Statistique de test : 0.9426178932189941
P-value : 1.7004279060358685e-16
Rsultat : Les donnes ne suivent pas une distribution normale

Interprtation :

  • Statistique de test : La statistique de test est comprise entre 0 et 1. Plus elle se rapproche de 1, plus il est probable que les donnes suivent une distribution normale. Dans mon cas, la statistique de test est d'environ 0.9426.

  • P-value : La p-value est un indicateur de la probabilit que les donnes proviennent d'une distribution normale. Plus la p-value est faible, plus il est peu probable que les donnes suivent une distribution normale. Dans mon cas, la p-value est trs proche de zro (1.7004e-16), ce qui signifie qu'elle est extrmement faible.

Rsultat : un niveau de signification de 0,05 ( = 0,05), ma p-value est bien infrieure ce seuil. Par consquent, je peux conclure que les donnes ne suivent pas une distribution normale.
